myFlix Angular

Angular built client side base on a Restful API and MongoDB database.

Features:

  • Register as new user.
  • Login with personal data.
  • Render list of all movies.
  • Show and hide movie's details.
  • Show and hide genre's details.
  • Show and hide director's details.
  • Add movies to favorite's list.
  • Remove movie from favorite's list.
  • Update profile
  • Delete profile
