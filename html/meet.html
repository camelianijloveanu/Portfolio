Meet App

Serverless, progressive web application (PWA) with React using a test-driven development (TDD) technique. The application uses the Google Calendar API to fetch upcoming events.

Features:

  • Filter events by city
  • Show and hide event's details
  • Specify numbers of events to be displayed
  • Usable when user is offline
  • Add app shortcut to the screen
  • View chart of upcoming events in a certain city
